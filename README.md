# Watch4Gether - GitHub Pages Version

Bu proje GitHub Pages için özel olarak hazırlanmış client-side React uygulamasıdır.

## 🌐 Canlı Site

**Ana Sayfa:** https://asdqwefz.github.io/inj/

**Room Sayfası:** https://asdqwefz.github.io/inj/room.html

## 🚀 Özellikler

- ✅ Çalışan butonlar (Create Room, Join Room)
- ✅ YouTube, Twitch, Kick video desteği
- ✅ Responsive tasarım
- ✅ Tamamen client-side (backend gerektirmez)

## 🛠️ Geliştirme

```bash
# Bağımlılıkları yükle
npm install

# Development server başlat
npm run dev

# Production build
npm run build
```

## 📝 Notlar

- Bu basit bir versiyondur
- Gerçek zamanlı senkronizasyon için backend gerekir
- Butonlar ve temel fonksiyonlar çalışır

## ⏰ GitHub Pages Güncelleme

GitHub Pages'in yeni dosyaları yayınlaması 5-10 dakika sürebilir.

Eğer butonlar çalışmıyorsa:
1. 5-10 dakika bekleyin
2. Tarayıcı önbelleğini temizleyin (Ctrl + Shift + Delete)
3. Gizli pencerede açın (Ctrl + Shift + N)
4. Sayfayı yenileyin (Ctrl + F5)
